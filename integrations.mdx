## Setting Up Third-Party Connections

Follow these steps to connect a third-party app like Slack.

<Steps>
  <Step title="Navigate to Integrations" icon="settings">
    Go to your dashboard and select **Integrations** from the sidebar.
  </Step>
  <Step title="Choose Provider" icon="plus">
    Search for and select the app, such as Slack.
  </Step>
  <Step title="Authorize Access" icon="key">
    Grant permissions by signing in with your provider account.
  </Step>
  <Step title="Configure Events" icon="sliders">
    Choose events like page updates or comments to trigger actions.
  </Step>
</Steps>

<Callout kind="tip">
Test the connection immediately after setup to verify notifications flow correctly.
</Callout>

## Using Webhooks for Automation

Webhooks allow external services to receive real-time events from your documentation space. Configure them via the dashboard or API.

<Tabs>
  <Tab title="Page Created" icon="plus">
    Triggered when a new page is published.
  </Tab>
  <Tab title="Page Updated" icon="edit">
    Fires on content changes or edits.
  </Tab>
  <Tab title="Comment Added" icon="message-circle">
    Notifies on new comments or replies.
  </Tab>
</Tabs>

Set up a webhook endpoint:

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  const express = require('express');
  const app = express();
  app.use(express.json());

  app.post('/webhook', (req, res) => {
    console.log('Event:', req.body.event);
    console.log('Page:', req.body.page);
    res.status(200).send('OK');
  });

  app.listen(3000);
  ```
  ```python
  from flask import Flask, request, jsonify
  app = Flask(__name__)

  @app.route('/webhook', methods=['POST'])
  def webhook():
      data = request.json
      print(f"Event: {data['event']}")
      print(f"Page: {data['page']}")
      return jsonify({'status': 'OK'}), 200

  if __name__ == '__main__':
      app.run(port=3000)
  ```
</CodeGroup>

## API Access for Custom Integrations

Build custom solutions using the REST API at `https://api.example.com/v1`.

<ParamField path="spaceId" param-type="string" required="true">
  Your documentation space identifier.
</ParamField>

<ParamField header="Authorization" param-type="string" required="true">
  Bearer `{YOUR_API_TOKEN}`.
</ParamField>

Example request to list integrations:

<Request tabs="cURL,JavaScript">
  ```bash
  curl -X GET https://api.example.com/v1/integrations \
    -H "Authorization: Bearer YOUR_API_TOKEN" \
    -H "Content-Type: application/json"
  ```
  ```javascript
  const response = await fetch('https://api.example.com/v1/integrations', {
    method: 'GET',
    headers: {
      'Authorization': 'Bearer YOUR_API_TOKEN',
      'Content-Type': 'application/json'
    }
  });
  const integrations = await response.json();
  ```
</Request>

<Response tabs="200">
  ```json
  {
    "integrations": [
      {
        "id": "slack_123",
        "provider": "slack",
        "status": "active",
        "events": ["page_updated"]
      }
    ]
  }
  ```
</Response>
