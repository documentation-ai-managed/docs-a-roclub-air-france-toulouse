- Added flight scheduling integration with real-time availability checks
- Introduced member profile customization options in the dashboard

- Enhanced search functionality across all documentation sections
- Optimized mobile responsiveness for booking forms

## Bug Fixes

<Update label="2024-08-10" description="v1.0.0" tags={["breaking", "feature"]}>

## New Features

- Initial release of the Aéroclub Air France - Toulouse documentation space
- Core sections: Quickstart, Authentication, and Flight Booking guides
- Integrated search and navigation for all club resources

## Breaking Changes

- Updated API base URL from `https://api.example.com/v0` to `https://api.example.com/v1`
- Authentication now requires JWT tokens instead of basic auth

## Migration Notes

Follow these steps to update your integrations:

<Steps>
  <Step title="Update Endpoints" icon="settings">
    Replace all `v0` paths with `v1` in your API calls.
  </Step>
  <Step title="Switch to JWT" icon="lock">
    Generate JWT tokens using the new auth flow at `https://auth.example.com/token`.
  </Step>
  <Step title="Test Integration" icon="play">
    Run your booking tests against the staging environment.
  </Step>
</Steps>

</Update>
