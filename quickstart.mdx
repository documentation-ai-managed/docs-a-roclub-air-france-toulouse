## Prerequisites

Before you begin, ensure you have the following:

<Callout kind="info">
- A modern web browser (Chrome, Firefox, Safari, or Edge)
- An email address for account verification
- Optional: Existing documentation files in Markdown or MDX format for import
</Callout>

## Log In to Your Space

Access your documentation workspace using one of these methods.

<Tabs>
  <Tab title="Web Browser" icon="globe">
    Go to `https://dashboard.example.com/login` and enter your credentials.
  </Tab>
  <Tab title="Magic Link" icon="magic-wand">
    Enter your email to receive a one-time login link.
  </Tab>
</Tabs>

<Callout kind="tip">
  Enable two-factor authentication in your account settings for added security.
</Callout>

## Basic Navigation and Workspace Setup

The sidebar provides quick access to pages, folders, and settings. Organize your content into folders by dragging pages.

- **Pages**: View and edit individual docs
- **Folders**: Group related documentation
- **Settings**: Configure workspace permissions and themes

Customize your workspace:

## Import Existing Content

Bring in your current docs seamlessly.

<Expandable title="Import Methods" default-open="true">
  Upload Markdown files directly or connect to GitHub repositories. Supported formats include `.md`, `.mdx`, and `.yaml`.

  <Steps>
    <Step title="File Upload">
      Drag and drop files into the import panel.
    </Step>
    <Step title="GitHub Sync">

    </Step>
  </Steps>
</Expandable>
